# Agenda Personal de Citas

##  Descripción del Caso de Uso
La *Agenda Personal de Citas* es un sistema diseñado para permitir que un usuario registre, organice y gestione sus citas personales de manera eficiente. El sistema facilita la programación de eventos con fecha, hora, lugar y una breve descripción, garantizando una experiencia clara y ordenada.

##  Objetivos del Sistema
- Permitir al usuario registrar nuevas citas personales.
- Facilitar la edición y eliminación de citas existentes.
- Mostrar una vista general de todas las citas programadas.
- Asegurar que la información se almacene de forma coherente y accesible.

##  Requerimientos
### **Requerimientos Funcionales**
1. El sistema debe permitir registrar nuevas citas.
2. El sistema debe permitir editar citas existentes.
3. El sistema debe permitir eliminar citas.
4. El sistema debe mostrar una lista completa de citas programadas.
5. El sistema debe validar que no se ingresen fechas vacías o inválidas.

### **Requerimientos No Funcionales**
1. La interfaz debe ser intuitiva y de fácil navegación.
2. El sistema debe registrar y mostrar la información en menos de 2 segundos.
3. La información debe almacenarse de forma segura y ordenada.

##  Plan de Pruebas Funcionales
| Caso de Prueba | Descripción | Resultado Esperado | Validación |
|----------------|-------------|---------------------|------------|
| CP01 | Registrar una cita nueva | La cita se guarda correctamente | ✔ Cumple |
| CP02 | Editar una cita existente | El sistema actualiza los campos modificados | ✔ Cumple |
| CP03 | Eliminar una cita | La cita se elimina de la lista | ✔ Cumple |

##  Propuesta de Mantenimiento
Se propone mantenimiento **perfectivo**, cuyo objetivo es mejorar la experiencia del usuario mediante optimizaciones en la interfaz, ampliación de funciones y mejoras en la usabilidad sin alterar el funcionamiento principal.
