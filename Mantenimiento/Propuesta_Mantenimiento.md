## 1. Mantenimiento Correctivo

# Objetivo:
Corregir errores que afecten el funcionamiento básico del sistema y eviten que el usuario realice acciones esenciales como registrar, editar o consultar citas.

# Acciones propuestas:

Solucionar fallos al registrar citas (por ejemplo, errores al guardar fecha, hora o datos en la base de datos).

Corregir errores en la modificación o eliminación de citas que no actualizan correctamente la información.

Resolver problemas en la visualización del calendario que impiden mostrar citas en ciertos días u horas.

Arreglar fallos de carga o bloqueo cuando hay demasiadas citas registradas.

Reparar incompatibilidades menores en navegadores específicos.

# Beneficio:
Garantiza que el sistema funcione correctamente, evitando interrupciones y asegurando que el usuario pueda gestionar sus citas sin errores.

## 2. Mantenimiento Adaptativo

# Objetivo:
Actualizar el sistema para que continúe funcionando correctamente a medida que cambien los requerimientos tecnológicos o los entornos donde se utiliza.

# Acciones propuestas:

Adaptar la aplicación para que sea compatible con nuevas versiones de navegadores web y sistemas operativos.

Implementar diseño responsive completo para garantizar uso óptimo en dispositivos móviles, tablets y pantallas de distintos tamaños.

Integrar la agenda con calendarios externos como Google Calendar u Outlook.

Actualizar librerías, frameworks o módulos que el sistema utilice para evitar vulnerabilidades o incompatibilidades.

Optimizar el sistema para soportar más usuarios o mayor volumen de citas sin fallos.

# Beneficio:
Permite que la agenda se mantenga funcional y accesible pese a cambios tecnológicos, asegurando continuidad a largo plazo.

## 3. Mantenimiento Perfectivo

# Objetivo:
Mejorar la experiencia del usuario, optimizar el rendimiento e implementar nuevas funcionalidades que aumenten la utilidad del sistema.

# Acciones propuestas:

Añadir notificaciones automáticas o recordatorios antes de cada cita (5, 10, 30 minutos).

Incorporar búsqueda avanzada por palabra clave, fecha, etiqueta o categoría.

Implementar etiquetas de colores para clasificar citas (médico, estudio, reuniones, personal, urgente).

Optimizar la interfaz para que sea más intuitiva, moderna y fácil de usar.

Agregar opción de exportar citas en formatos PDF o CSV.

Mejorar la velocidad de carga del calendario y la navegación, especialmente cuando existen más de 100 citas registradas.

# Beneficio:
Aumenta la satisfacción del usuario, mejora el rendimiento del sistema y agrega funciones que incrementan la productividad y valor de la aplicación.
